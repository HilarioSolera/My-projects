Instrucciones para ejecutar el compilador:
1. Asegúrate de tener instalado Java en tu máquina.
2. Compila el programa en tu entorno de desarrollo (NetBeans o línea de comandos).
3. Ejecuta el programa proporcionando el archivo Python que deseas analizar.
4. Revisa el archivo de errores generado para cualquier problema encontrado.
5. Observa el resumen de comentarios y tokens al final del archivo de errores.
